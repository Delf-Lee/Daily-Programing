# Daily-Programing

## Question

### DP001
문자열 배열(string array)이 주어지면, 제일 긴 공통된 접두사(prefix)의 길이를 찾으시오.

- Input: [“apple”, “apps”, “ape”]  
output: 2 // “ap”
- Input: [“hawaii”, “happy”]  
Output: 2 // “ha”
- Input: [“dog”, “dogs”, “doge”]  
Output: 3 // “dog”

### DP002
정수 배열(int array)과 정수 N이 주어지면, N번째로 큰 배열 원소를 찾으시오.

- Input: [-1, 3, -1, 5, 4], 2  
Output: 4
- Input: [2, 4, -2, -3, 8], 1  
Output: 8
- Input: [-5, -3, 1], 3  
Output: -5
